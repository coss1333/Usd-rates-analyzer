Пошаговая инструкция по использованию программы для анализа изменения стоимости USD
-------------------------------------------------------------------------------

1️⃣ Установка Python
   - Скачай и установи Python с сайта: https://www.python.org/downloads/
   - При установке отметь галочку “Add Python to PATH”.

2️⃣ Подготовка папки
   - Распакуй архив "usd_rates_analyzer.zip" в удобное место (например, на рабочий стол).
   - В папке должны быть три файла:
       • usd_rates_tracker.py
       • requirements.txt
       • instruction_usd_rates.txt

3️⃣ Установка зависимостей
   - Открой терминал (или PowerShell) в этой папке.
   - Выполни команду:
       pip install -r requirements.txt

4️⃣ Запуск программы
   - В том же терминале введи:
       python usd_rates_tracker.py

5️⃣ Что делает программа
   - Показывает изменение курса USD относительно 10 основных валют (EUR, GBP, JPY, CHF, CAD, AUD, NZD, CNY, SEK, NOK).
   - Сравнивает вчерашние и сегодняшние значения.
   - Выводит таблицу с процентным изменением.

6️⃣ Источник данных
   - Программа использует бесплатный API: https://api.exchangerate.host
   - Регистрация и API-ключ не требуются.
